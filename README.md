# Local File Storage Management System

A File Storage Management System developed using HTML, CSS, JavaScript and Python.

## Features

- Upload Files
- Delete Files
- Preview Files
- Download Files
- Copy Files Between Storage Locations
- Activity Logs
- Responsive UI

## Technologies

- HTML5
- CSS3
- JavaScript
- Python
- Tkinter

## Project Structure

index.html
styles.css
app.js
file_storage.py

## How to Run

### Web Version

Open index.html in your browser.

### Python Version

python file_storage.py
